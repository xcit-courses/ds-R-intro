What is this course about?

  1. Getting started (background and setup)
  2. Programming, in R
  3. Introduction to the tidyverse
  4. Importing data into R
  5. Data visualization Design
  6. Data visualization using ggplot2
  7. Data tidying and transformations
  8. Data analysis (modeling, statistics, ML)
  9. Workflows & best practices (reproducibility)
  10. …anything else?

Why this course?

Data science workflow

Programming is the “glue” that binds all of these steps.

Freedom and empowerment

  • You are restricted when using “tools” developed by others;

  • If you learn to create your own “tools”, the possibilities become endless.

  • You will gain in efficiency and freedom (you can delegate work to computers).

  • You will better understand the digital world.

Learning by doing

This is a hands-on course!
